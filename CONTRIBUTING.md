# Contributing to Links I Like

Thanks for your interest in contributing! This is a personal collection of links I share, but I welcome improvements and suggestions.

## How to Contribute

### Found a Broken Link?
1. Open an issue with the broken link and its location
2. I'll update or remove it

### Have a Suggestion?
1. Open an issue describing your suggestion
2. If it's about organization or presentation, feel free to submit a PR

### Want to Submit a Link?
While this is primarily my personal collection, if you have something absolutely amazing that fits the categories, feel free to:
1. Open an issue with the link and why it's awesome
2. I'll review and potentially add it

## Guidelines

- Keep descriptions concise and helpful
- Make sure links are accessible and not behind paywalls (unless specifically noted)
- Categorize appropriately
- Include dates when shared

## Pull Request Process

1. Fork the repo
2. Make your changes
3. Test that all links work
4. Submit a PR with a clear description

## Code of Conduct

Be respectful and constructive. This is a learning resource for everyone.

Thanks!
